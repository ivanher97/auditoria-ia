# Landing Asesoría en IA y Automatización

Landing page profesional y moderna para servicios de consultoría y auditoría en Inteligencia Artificial, Big Data y Automatización de Procesos.

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica.
- **CSS3 (Vanilla)**: Diseño moderno, efectos glassmorphism, orbes de brillo dinámicos y transiciones suaves.
- **JavaScript**: Lógica interactiva en el frontend.

## Estructura del Proyecto

```text
├── index.html     # Estructura principal de la landing page
├── styles.css     # Estilos CSS con temática oscura y efectos visuales modernos
└── script.js      # Scripts para interactividad y efectos dinámicos
```
